Motion UI Docs

Welcome!

These are the docs for the alpha version of the Foundation for Apps Motion UI. The goal here is to allow you to rapidly prototype animated elements that integrate seamlessly into your site!

{{#markdown}} #### Base Motion Quickly addding motion to an element is easy! 1. Use the attribute `data-motion` to describe how we want an element to move. 2. Use the class `motion` to trigger the animation. 3. [There's no step 3!](http://youtu.be/6uXJlX50Lj8) {{/markdown}}

{{#markdown}} #### Modifiers Don't want an out-of-the-box animation? No problemo! You have several modifiers at your disposal to get exactly the animation you want! {{/markdown}}
{{#markdown}} ##### Speed You have three speeds to choose from. By default, animations will execute in **700ms**. You can pass `slow` or `fast` modifiers to change the speed to **1.5s** or **300ms** respectively. ###### Markup ```html

{{#markdown}} #### Build with Mixins You can add motion directly with our Sass mixins for more semantic code and a finer level of control. {{/markdown}}
{{#markdown}} ##### Motion Variables ```scss $transition-normal-duration: 700ms; $motion-class: "motion"; $motion-default-origin: left; $slide-default-dir: left; $include-html-motion-attributes: true; $auto-adjust-opacity: true; $motion-default-property: transform; $motion-default-timing: ease; $motion-default-delay: 0; $motion-slow-duration: 1.4s; $motion-default-duration: 700ms; $motion-fast-duration: 300ms; ``` ##### Basic Mixin Usage ```scss // Include a transition in your element // // transition-name: name of a transition mixin. // [ slide | fade | hinge | scale | spin ] @include transition-name(); ``` ###### Global Arguments **$duration**
Default: `$motion-default-duration // 300ms`
*Length of time in which animation executes*
Options: `` **$timing**
Default: `$motion-default-timing // ease`
*Set the motion timing function*
Options: `` **$delay**
Default: `$motion-default-delay // 0`
*Amount of time to elapse before the animation executes.*
Options: `` **$fade** (does not apply to prefab animations)
Default: `true`
*Control if element fades in/out with the animation*
Options: `true`, `false`
##### Slide ```scss // Slide with default settings @include slide(); // Slide from top without fading with a 300ms delay @include slide(top, false, $delay:300ms); // Slide in from top with a linear timing function @include slide($from:right, $timing:linear); ``` ###### Arguments **$from**
Default: `left`
*Defines the element's slide origin.*
Options: `top`, `right`, `bottom`, `left`
##### Fade ```scss // Fade with default settings @include fade(); // Fade from half opacity to fully opaque @include fade($from:0.5, $to:1); ``` **$from**
Default: `0`
*Defines the initial opacity.*
Options: `` **$to**
Default: `1`
*Defines the element destination opacity*
Options: ``
##### Hinge ```scss // Hinge with default settings @include hinge(); // Hinge from top to bottom on a center axis @include hinge($from:top, $axis:center) // Hinge from front to back @include hinge($turn-origin:from-front) ``` **$from**
Default: `left`
*Defines the elements directional origin*
Options: `top`, `right`, `bottom`, `left` **$axis**
Default: `edge`
*Defines where the element turns form*

*Adjust the `perspective()` transform function*

*Define the z-orientation the element starts from*
